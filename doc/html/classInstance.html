Detailed Description

This class stores the initial mapping that serves as input for the Online Routing and Spectrum Allocation problem.

This consists of a topology graph where some slices of some edges are already occupied by some given demands.

enum Instance::Metric

Enumerates the possible output policies to be used.

Enumerator
METRIC_ONE 

Sum over demands of last slot used for each demand.

METRIC_ONE_P 

Sum over edges of the last slot used on each edge.

METRIC_TWO 

Sum of hops.

METRIC_FOUR 

Sum of path lengths.

METRIC_EIGHT 

Last slot used overall.

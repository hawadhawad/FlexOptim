Detailed Description

This class contains all the information needed for the creation of an instance.

It stores input/output file paths, execution parameters (such as the chosen method used to solve the problem), and control parameters (such as max number of iterations).

◆ PreprocessingLevel

enum Input::PreprocessingLevel

Enumerates the possible levels of applying a preprocessing step fo reducing the graphs before optimization is called.

Enumerator
PREPROCESSING_LVL_NO 

Only remove arcs that do not fit the demand load.

PREPROCESSING_LVL_PARTIAL 

Previous steps + look for arcs that would induce length violation and arcs whose neighboors cannot forward the demand.

PREPROCESSING_LVL_FULL 

Previous steps recursively until no additional arc can be removed.

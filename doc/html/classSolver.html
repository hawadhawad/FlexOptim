Detailed Description

This class specifies which MIP solver to be used and stores the new arc maps used for identifying arcs after preprocessing has erased some arcs.

The methods needed for solving the Routing and Spectrum Allocation problem via CPLEX.

Note
It uses the LEMON library to build the arc maps and CPLEX concert library to build expressions and constraints.

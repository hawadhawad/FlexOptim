Detailed Description

This is an abstract class modelling a MIP solver.

Derived classes need to specify how the RSA formulation is solved, i.e., which MIP solved is used.

Note
It uses the LEMON library to build the arc maps and CPLEX concert library to build expressions and constraints.

enum Solver::Status

Enumerates the possible algorithm status according to the current model and solution in hand.

Enumerator
STATUS_UNKNOWN 

The algorithm has no information about the solution of the model.

STATUS_FEASIBLE 

The algorithm found a feasible solution that may not necessarily be optimal.

STATUS_OPTIMAL 

The algorithm found an optimal solution.

STATUS_INFEASIBLE 

The algorithm proved the model infeasible; that is, it is not possible to find a feasible solution.

STATUS_UNBOUNDED 

The algorithm proved the model unbounded.

STATUS_INFEASIBLE_OR_UNBOUNDED 

The model is infeasible or unbounded.

STATUS_ERROR 

An error occurred.

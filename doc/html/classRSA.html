Detailed Description

This class stores the input needed for solving the Routing and Spectrum Allocation problem.

This consists of an initial mapping and a set of demands to be routed. A graph associated to this initial mapping is built as well as an extended graph for each demand to be routed.

Note
It uses the LEMON library to build the associated graphs.

Member Enumeration Documentation

◆ Status

enum RSA::Status

Enumerates the possible algorithm status according to the current model and solution in hand.

Enumerator
STATUS_UNKNOWN 

The algorithm has no information about the solution of the model.

STATUS_FEASIBLE 

The algorithm found a feasible solution that may not necessarily be optimal.

STATUS_OPTIMAL 

The algorithm found an optimal solution.

STATUS_INFEASIBLE 

The algorithm proved the model infeasible; that is, it is not possible to find a feasible solution.

STATUS_UNBOUNDED 

The algorithm proved the model unbounded.

STATUS_INFEASIBLE_OR_UNBOUNDED 

The model is infeasible or unbounded.

STATUS_ERROR 

An error occurred.

Detailed Description

This class identifies a demand.

A demand is defined by its id, its source and target node, its load (i.e., how many slices it requests) and its maximal length (i.e., how long can be its routing path). It also can be already routed or not.

Note
The id of each Demand is considered to be in the range [0, ..., n-1].

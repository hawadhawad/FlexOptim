\doxysubsection{Detailed Description}
This class identifies a demand. 

A demand is defined by its id, its source and target node, its load (i.\+e., how many slices it requests) and its maximal length (i.\+e., how long can be its routing path). It also can be already routed or not. \begin{DoxyNote}{Note}
The id of each \mbox{\hyperlink{classDemand}{Demand}} is considered to be in the range \mbox{[}0, ..., n-\/1\mbox{]}. 
\end{DoxyNote}

\doxysubsection{Detailed Description}
This class stores the input needed for solving the Routing and Spectrum Allocation problem. 

This consists of an initial mapping and a set of demands to be routed. A graph associated to this initial mapping is built as well as an extended graph for each demand to be routed. \begin{DoxyNote}{Note}
It uses the L\+E\+M\+ON library to build the associated graphs. 
\end{DoxyNote}

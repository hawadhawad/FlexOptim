\doxysubsection{Detailed Description}
This class identifies a \mbox{\hyperlink{classSlice}{Slice}} of a \mbox{\hyperlink{classPhysicalLink}{Physical\+Link}}. 

~\newline
 A \mbox{\hyperlink{classDemand}{Demand}} may be routed through a \mbox{\hyperlink{classSlice}{Slice}}. This class allows to identify, specify and verify which \mbox{\hyperlink{classDemand}{Demand}} is routed (or not) through the slice.

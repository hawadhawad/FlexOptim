\doxysubsection{Detailed Description}
This class identifies an specific slice in the frequency spectrum of a \mbox{\hyperlink{classPhysicalLink}{Physical\+Link}}. 

A \mbox{\hyperlink{classDemand}{Demand}} that is routed through a \mbox{\hyperlink{classPhysicalLink}{Physical\+Link}} must be assigned to n continuous slices, where n is the demand\textquotesingle{}s load. This class allows to identify, specify and verify which \mbox{\hyperlink{classDemand}{Demand}} is assigned (or not) through a slice.

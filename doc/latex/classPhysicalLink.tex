\doxysubsection{Detailed Description}
This class identifies a link on the physical network. 

It corresponds to an edge in the topology graph and is defined by its id, a source and a target node, its length in the physical network, and a cost. A edge also has a frequency spectrum that is splitted into a given number of slices. \begin{DoxyNote}{Note}
The id of each \mbox{\hyperlink{classPhysicalLink}{Physical\+Link}} is considered to be in the range \mbox{[}0, ..., n-\/1\mbox{]}. ~\newline
 
\end{DoxyNote}
